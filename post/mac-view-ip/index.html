苹果电脑命令行查看本机的公网IP和内网IP

查看本机公网IP命令


curl ifconfig.me

json格式化


curl ipinfo.io/json

还可以用wget


wget http://ipecho.net/plain -O - -q ; echo
wget http://observebox.com/ip -O - -q ; echo

查看本机内网IP


ifconfig

（ 找到名为 en0（本地以太网）的网络，inet 就是本机当前ip地址 ）
